# Macros

**Macro** management: create, delete and configure. Subsections: **New Macro**, **Delete Macro**, **Macros** (list).

---

## New Macro

Button that opens a dialog to **create a new macro**. You are asked for a name; the new macro appears in the list and can be configured (hotkey, actions).

---

## Delete Macro

Button to **delete the selected macro** in the list. Confirmation is required before deleting.

---

## Macros (list)

List of all macros in your profile. When you select a macro, the right panel shows:

- Macro **name**.
- **Hotkey** assigned.
- **Actions** the macro runs (sequence of commands/actions).

Each macro can be edited to change key and action sequence. Exact options depend on the action type (movement, spell, item use, etc.).
