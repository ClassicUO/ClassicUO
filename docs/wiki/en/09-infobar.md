# Infobar

**Info bar** options – line or area that shows character data (health, mana, stamina, etc.).

| Option | Type | Description |
|--------|------|-------------|
| **Active Info Bar** | Checkbox/area | Enable the infobar and choose which is active (if there are several). |
| **Select type infobar** | List/buttons | Choose infobar type or style (classic, compact, etc.). |

Labels come from `GetInfoBars`. The infobar shows quick player info without opening gumps.
