# Combat & Spells

| Option | What it does |
|--------|--------------|
| **Hold tab for combat** | Holding Tab activates combat mode. |
| **Query before attack** | Asks before attacking (e.g. gray targets). |
| **Query before beneficial acts on murderers/criminals/gray** | Asks before casting beneficial spells on those targets. |
| **Enable overhead spell format** | Overhead spell text format. |
| **Enable overhead spell hue** | Hue of overhead spell text. |
| **Single click for spell icons** | Single click on spell icons (instead of double). |
| **Show buff duration on old style buff bar** | Buff duration on the old-style buff bar. |
| **Enable fast spell hotkey assigning** | Quick hotkey assignment for spells. |
| **Ctrl + Alt + Click spell icon** | Opens gump to set hotkey on that icon. |
| **Innocent / Beneficial / Friend / Harmful / Criminal / Neutral / Can be attacked / Murderer / Enemy** | Hue by notoriety for spells and targets. |
| **Spell overhead format** | Placeholders e.g. `{power}` for powerword, `{spell}` for spell name. |
