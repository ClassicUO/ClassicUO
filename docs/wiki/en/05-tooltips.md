# Tooltips

**Tooltip** options (text that appears when hovering over items, mobiles, etc.).

| Option | Type | Description |
|--------|------|-------------|
| **Enable tooltips** | Checkbox | Enables tooltips in game. |
| **Tooltip delay** | Slider/field | Delay in ms before the tooltip appears after hovering. Increase to reduce "jumping" when moving the mouse; decrease for faster response. |
