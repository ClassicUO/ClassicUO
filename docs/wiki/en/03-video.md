# Video

**Video** options: game window, zoom, lighting, shadows and misc. Subsections: **Game window**, **Zoom**, **Lighting**, **Misc**, **Shadows**.

---

## Game window

| Option | Type | Description |
|--------|------|-------------|
| **FPS Cap** | Slider | Maximum FPS limit. |
| **Enable VSync** | Checkbox | Vertical sync; limits FPS to monitor refresh rate. |
| **Unlimited FPS** | Checkbox | Removes FPS cap (may increase GPU usage). |
| **Reduce FPS when game is not in focus** | Checkbox | Lowers FPS when window is not focused to save resources. |
| **Always use fullsize game world viewport** | Checkbox | World viewport always uses full size. |
| **Fullscreen window** | Checkbox | Fullscreen window mode. |
| **Lock game world viewport position/size** | Checkbox | Locks viewport position and size. |
| **Viewport position X / Y** | Slider | Viewport X and Y position. |
| **Viewport width / height** | Slider | Viewport width and height. |

---

## Zoom

| Option | Type | Description |
|--------|------|-------------|
| **Default zoom** | Slider | Initial zoom level. |
| **Enable zooming with ctrl + mousewheel** | Checkbox | Zoom with **Ctrl** + mouse wheel. |
| **Return to default zoom after ctrl is released** | Checkbox | When releasing Ctrl, returns to default zoom. |

---

## Lighting

| Option | Type | Description |
|--------|------|-------------|
| **Alternative lights** | Checkbox | Uses alternative lighting system. |
| **Custom light level** | Checkbox | Allows setting light level manually. |
| **Light level** | Slider | Light level (when custom is on). |
| **Light level type** | Combo | **Absolute** or **Minimum**. |
| **Dark nights** | Checkbox | Darker nights. |
| **Colored lighting** | Checkbox | Colored lighting. |

---

## Misc (Video)

| Option | Type | Description |
|--------|------|-------------|
| **Enable death screen** | Checkbox | Special screen when dead. |
| **Black and white mode while dead** | Checkbox | Black and white screen when dead. |
| **Run mouse in separate thread** | Checkbox | Mouse in separate thread (may improve responsiveness). |
| **Aura on mouse target** | Checkbox | Aura on target under cursor. |
| **Animated water effect** | Checkbox | Animated water. |
| **Visual style** | Combo | **Classic (2D)** or **Enhanced (3D-like)**. |

---

## Shadows

| Option | Type | Description |
|--------|------|-------------|
| **Enable shadows** | Checkbox | Enables shadows in the world. |
| **Rock and tree shadows** | Checkbox | Shadows on rocks and trees. |
| **Terrain shadow level** | Slider | Terrain shadow level. |
