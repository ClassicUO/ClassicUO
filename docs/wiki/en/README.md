# Wiki – Client Options (Modern Options)

**Item-by-item** guide to **Modern Options** (client options window). Each section matches the left sidebar menu and the right-panel subsections. Each option is explained in detail: what it does, how to configure it, and when relevant, requirements or warnings.

**How to open Modern Options:** In game, open **Options** (or your configured shortcut). Use the search bar at the top of the window to find options by name.

## How to use the wiki

- Each page describes **each option** in that section: name, type (checkbox, slider, combo, color picker, etc.), what it does and how to configure it.
- Options are grouped by **subsection** (left panel within each category).
- Settings are saved automatically to your profile when you change them; no need to "Save" manually.
- Some sections include a **Detail per option** area with longer explanations and usage tips.
