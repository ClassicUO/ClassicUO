# Containers

**Container** options (backpacks, chests, etc.): how they are displayed and open behaviour.

| Option | Type | Description |
|--------|------|-------------|
| **Layout / style** | Combo/checkboxes | Gump type (original, grid, etc.) and visual options. |
| **Scale / opacity** | Slider | Container size and opacity. |
| **Others** | Various | Options from `GetContainers` (Language): open at a given position, remember position, etc. |

Distinct from **Grid Loot** (in General → Gumps): Grid Loot is the grid loot gump for corpses; Containers refers to bag/chest gumps in general.
