# Sound

**Sound** and **music** options. There are no subsections in the left panel; all options appear on one page.

| Option | Type | Description |
|--------|------|-------------|
| **Enable sound** | Checkbox | Enables in-game sound effects. |
| **Volume** (sound) | Slider (0–100) | Master volume for sound effects. |
| **Enable music** | Checkbox | Enables background music. |
| **Volume** (music) | Slider (0–100) | Music volume. |
| **Enable login page music** | Checkbox | Music on the login page. |
| **Play footsteps** | Checkbox | Footstep sounds when walking/running. |
| **Combat music** | Checkbox | Music during combat. |
| **Play sound when UO is not in focus** | Checkbox | Game continues to play sound/music when the window is in the background. |
