# Nameplate Options

**Nameplate** options (names and info above mobiles/players). Includes creating and editing **overhead profiles**.

| Option | Type | Description |
|--------|------|-------------|
| **New entry** | Button | Creates a new nameplate profile (custom name). You can set conditions (mobile name, notoriety, etc.) and what to show. |
| **Delete entry** | Button | Deletes the currently selected nameplate profile (with confirmation). |
| **Entry list** | Left panel | Each entry is a profile. Selecting one shows that profile's rules and options in the right panel (who to show, text, color, etc.). |

Profiles allow e.g. showing names only on enemies, or only on party, with different colors. Exact options depend on `NameOverheadAssignControl` and `GetNamePlates`.
