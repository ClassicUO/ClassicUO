# Speech

**Speech** options (chat, text substitutions, filters, speech macros). Includes multiple subsections in the left panel depending on the gump build.

| Option | Type | Description |
|--------|------|-------------|
| **Channel activation** | Various | Enable/disable chat channels (general, party, guild, etc.). |
| **Substitutions** | Fields | Text to replace when sending (e.g. abbreviations that expand to phrases). |
| **Filters** | Checkboxes/list | Filter messages by type or source. |
| **Speech macros** | Buttons/list | Shortcuts that send predefined phrases in chat. |

Exact labels depend on the client Language (Speech). Behaviour: define what you say, where you say it, and which shortcuts trigger messages.
