# Action Bar

Customisable action bar (buttons for spells, items, macros).

| What you find | What it does |
|---------------|--------------|
| **+ Add** | Adds a new button to the bar. |
| **Buttons** | Each button can be set to run an action (spell, item, macro, etc.). |

The bar can be moved and buttons are user-defined (drag icons, choose actions).
