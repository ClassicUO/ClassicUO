# TazUO Specific

**TazUO**-specific options: grid containers, journal, modern paperdoll, nameplates, mobiles and misc. Subsections in the left panel.

---

## Grid Containers

| Option | Type | Description |
|--------|------|-------------|
| **Enable Grid Containers** | Checkbox | Containers (bags, chests) open in grid layout instead of classic. |
| **Grid Container Scale** | Slider (50–200%) | Scale of items in the grid. |
| **Also Scale Items** | Checkbox | Applies scale to item icons. |
| **Grid Item Border Opacity** | Slider (0–100) | Item border opacity. |
| **Border Color** | Color picker | Border color. |
| **Container Opacity** | Slider | Container background opacity. |
| **Background Color** | Color picker | Background color. |
| **Use Containers Hue** | Checkbox | Use container hue. |
| **Search Style** | Combo | **Only Show** or **Highlight** when searching. |
| **Enable Container Preview** | Checkbox | Preview on hover (see tooltip). |
| **Make Anchorable** | Checkbox | Allows anchoring the grid gump. |
| **Container Style** | Combo | Border style. |
| **Hide Borders** | Checkbox | Hides gump borders. |
| **Default Grid Rows / Columns** | Slider (1–20) | Default rows/columns. |
| **Grid Highlight Settings** | Button | Opens grid highlight menu. |
| **Grid Highlight Size** | Slider (1–5) | Highlight size. |

---

## Journal

| Option | Type | Description |
|--------|------|-------------|
| **Max Journal Entries** | Slider (100–2000) | Max journal lines stored. |
| **Journal Opacity** | Slider (0–100) | Background opacity. |
| **Journal Background Color** | Color picker | Background color. |
| **Journal Style** / **Hide Borders** / **Hide Timestamp** | Combo/Checkbox | Style and visibility. |
| **Make Anchorable** | Checkbox | Journal can be anchored. |
| **Journal Messages Only In Journal Box** | Checkbox | System messages only in journal. |

---

## Modern Paperdoll

| Option | Type | Description |
|--------|------|-------------|
| **Enable Modern Paperdoll** | Checkbox | Uses modern paperdoll instead of classic. |
| **Paperdoll Hue** / **Durability Bar Hue** | Color picker | Colors. |
| **Show Durability Bar Below** | Slider (1–100%) | Show durability bar below X%. |
| **Paperdoll Anchor** | Checkbox | Paperdoll can be anchored. |

---

## Nameplates (TazUO) / Mobiles (TazUO) / Misc (TazUO)

Nameplates (HP on nameplate, opacity, hide when full health, etc.), Mobiles (damage colors, overhead, health bars, hidden player opacity, auto-follow, etc.), Misc (system chat, auto avoid, buff gump, main window background, health indicator, spell icon scale, hotkeys on icons, etc.). See the in-game options for full labels.
