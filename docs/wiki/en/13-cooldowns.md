# Cooldown bars

**Cooldown bar** options (TazUO): position and when they appear.

| Option | Type | Description |
|--------|------|-------------|
| **Custom Cooldown Bars** | Area/title | Cooldown bar section. |
| **Position X / Y** | Numeric field | Screen position (pixels) where the cooldown bar is drawn. |
| **Use last moved bar position** | Checkbox | Save and use the last position the bar was dragged to. |
| **Conditions** | Area | Conditions that determine when the bar appears (e.g. in warmode, when disarmed). |
| **Add Condition** | Button | Adds a new condition; reopens Modern Options on the Cooldowns page to configure it. |

Useful to see disarm, swing or other cooldowns in a fixed place on screen.
