# Counters

**Counter** options (e.g. mana, health, stamina shown as numbers or bars somewhere in the UI).

| Option | Type | Description |
|--------|------|-------------|
| **Activation** | Checkbox | Enable or disable the counter system. |
| **Type / position** | Combo/slider | Where and how counters are shown (bar, number, screen position). |
| **Values** | Slider/field | Thresholds or default values (e.g. warn when mana < X). |

Labels depend on `GetCounters`. Used to track resources (HP, mana, stam) visibly.
