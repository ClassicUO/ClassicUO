# Cooldown bars

Opções das **barras de cooldown** (TazUO): posição e condições em que aparecem.

| Opção | Tipo | Descrição |
|-------|------|-----------|
| **Custom Cooldown Bars** | Área/título | Secção das barras de cooldown. |
| **Position X / Y** | Campo numérico | Posição no ecrã (em pixels) onde a barra de cooldown é desenhada. |
| **Use last moved bar position** | Checkbox | Guardar e usar a última posição para onde a barra foi arrastada. |
| **Conditions** | Área | Condições que determinam quando a barra aparece (ex.: em warmode, quando desarmado). |
| **Add Condition** | Botão | Adiciona uma nova condição; reabre o Modern Options na página Cooldowns para a configurar. |

Útil para ver cooldowns de disarm, swing ou outras acções num lugar fixo do ecrã.
