# Combat & Spells

| Opção | O que faz |
|-------|-----------|
| **Hold tab for combat** | Tab mantido premido ativa modo combate. |
| **Query before attack** | Pergunta antes de atacar (ex.: alvos cinzentos). |
| **Query before beneficial acts on murderers/criminals/gray** | Pergunta antes de magias benéficas nesses alvos. |
| **Enable overhead spell format** | Formato do texto de magia no overhead. |
| **Enable overhead spell hue** | Cor (hue) do texto de magia no overhead. |
| **Single click for spell icons** | Um clique nos ícones de magia (em vez de duplo). |
| **Show buff duration on old style buff bar** | Duração dos buffs na barra antiga. |
| **Enable fast spell hotkey assigning** | Atribuição rápida de hotkeys às magias. |
| **Ctrl + Alt + Click spell icon** | Abre gump para definir hotkey nesse ícone. |
| **Innocent / Beneficial / Friend / Harmful / Criminal / Neutral / Can be attacked / Murderer / Enemy** | Cores (hue) por notoriety para magias e alvos. |
| **Spell overhead format** | Placeholders: e.g. `{power}` para powerword, `{spell}` para nome da magia. |
