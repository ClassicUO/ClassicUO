# Counters

Opções dos **contadores** (ex.: mana, vida, stamina exibidos como números ou barras em algum lugar da UI).

| Opção | Tipo | Descrição |
|-------|------|-----------|
| **Activação** | Checkbox | Activar ou desactivar o sistema de contadores. |
| **Tipo / posição** | Combo/slider | Onde e como os contadores são mostrados (barra, número, posição no ecrã). |
| **Valores** | Slider/campo | Limiares ou valores por defeito (ex.: avisar quando mana &lt; X). |

As etiquetas dependem de `GetCounters`. Servem para acompanhar recursos (HP, mana, stam) de forma visível.
