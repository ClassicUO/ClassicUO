# Wiki – Opções do cliente (Modern Options)

Guia **item a item** das opções do **Modern Options** (janela de opções do cliente). Cada secção corresponde ao menu lateral esquerdo e às subsecções no painel direito. Cada opção é explicada em detalhe: o que faz, como configurar e, quando relevante, requisitos ou avisos.

**Como abrir o Modern Options:** em jogo, abra **Options** (ou o atalho configurado). Use a barra de pesquisa no topo da janela para encontrar opções por nome.

## Como usar a wiki

- Cada página descreve **cada opção** da respectiva secção: nome, tipo (checkbox, slider, combo, color picker, etc.), o que faz e como configurar.
- As opções estão agrupadas por **subsecção** (painel esquerdo dentro de cada categoria).
- As definições são guardadas automaticamente no perfil ao alterar; não é necessário "Guardar" manualmente.
- Em algumas secções há uma área **Detalhe por opção** com explicações mais longas e dicas de uso.
