# Action Bar

Barra de ações personalizável (botões para magias, itens, macros).

| O que encontras | O que faz |
|-----------------|-----------|
| **+ Add** | Adiciona um novo botão à barra. |
| **Botões** | Cada botão pode ser configurado para executar uma ação (magia, item, macro, etc.). |

A barra pode ser movida e os botões são definidos pelo utilizador (arrastar ícones, escolher ações).
