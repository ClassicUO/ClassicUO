# TazUO Specific

Opções específicas do **TazUO**: grid containers, journal, paperdoll moderno, nameplates, mobiles e misc. Subsecções no painel esquerdo.

---

## Grid Containers

| Opção | Tipo | Descrição |
|-------|------|-----------|
| **Enable Grid Containers** | Checkbox | Contentores (sacos, baús) abrem em layout de grelha em vez do clássico. |
| **Grid Container Scale** | Slider (50–200%) | Escala dos itens na grelha. |
| **Also Scale Items** | Checkbox | Aplica a escala aos ícones dos itens. |
| **Grid Item Border Opacity** | Slider (0–100) | Opacidade da borda dos itens. |
| **Border Color** | Color picker | Cor da borda. |
| **Container Opacity** | Slider | Opacidade do fundo do contentor. |
| **Background Color** | Color picker | Cor de fundo. |
| **Use Containers Hue** | Checkbox | Usar a cor (hue) do contentor. |
| **Search Style** | Combo | **Only Show** ou **Highlight** ao pesquisar. |
| **Enable Container Preview** | Checkbox | Pré-visualização ao passar o rato (ver tooltip). |
| **Make Anchorable** | Checkbox | Permite ancorar o gump de grid. |
| **Container Style** | Combo | Estilo da borda (enum BorderStyle). |
| **Hide Borders** | Checkbox | Esconde as bordas do gump. |
| **Default Grid Rows / Columns** | Slider (1–20) | Número de linhas/colunas por defeito. |
| **Grid Highlight Settings** | Botão | Abre menu de destaque na grelha. |
| **Grid Highlight Size** | Slider (1–5) | Tamanho do destaque. |

## Journal

| Opção | Tipo | Descrição |
|-------|------|-----------|
| **Max Journal Entries** | Slider (100–2000) | Número máximo de linhas guardadas no journal. |
| **Journal Opacity** | Slider (0–100) | Opacidade do fundo. |
| **Journal Background Color** | Color picker | Cor de fundo. |
| **Journal Style** | Combo | Estilo da borda. |
| **Journal Hide Borders** | Checkbox | Esconde bordas. |
| **Hide Timestamp** | Checkbox | Esconde a hora em cada linha. |
| **Make Anchorable** | Checkbox | Journal pode ser ancorado. |
| **Journal Messages Only In Journal Box** | Checkbox | Mensagens de sistema só no journal (não noutros sítios). |

## Modern Paperdoll

| Opção | Tipo | Descrição |
|-------|------|-----------|
| **Enable Modern Paperdoll** | Checkbox | Usa o paperdoll moderno em vez do clássico. |
| **Paperdoll Hue** | Color picker | Cor/tema do paperdoll. |
| **Durability Bar Hue** | Color picker | Cor da barra de durabilidade. |
| **Show Durability Bar Below** | Slider (1–100%) | Mostrar barra de durabilidade abaixo de X% da durabilidade. |
| **Paperdoll Anchor** | Checkbox | Paperdoll pode ser ancorado. |

## Nameplates (TazUO)

| Opção | Tipo | Descrição |
|-------|------|-----------|
| **Nameplates Also Act As Health Bars** | Checkbox | Nameplates mostram também barra de vida. |
| **HP Opacity** | Slider (0–100) | Opacidade da barra de HP no nameplate. |
| **Hide Nameplates If Full Health** | Checkbox | Esconde nameplate quando vida está cheia. |
| **Only In Warmode** | Checkbox | (Dentro de Hide If Full Health.) Só esconde em warmode. |

| **Background Opacity** | Slider | Opacidade do fundo. |

## Mobiles (TazUO)

Cores de dano, overhead, barras de vida e outros comportamentos relacionados a mobiles.

| Opção | Tipo | Descrição |
|-------|------|-----------|
| **Damage To Self / Others / Pets / Allies / Last Attack** | Color picker | Cor do texto de dano para si, outros, pets, aliados, último ataque. |
| **Display Party Chat Over Player Heads** | Checkbox | Chat de grupo aparece em overhead nos jogadores. |
| **Overhead Text Width** | Slider (0–600) | Largura máxima do texto em overhead. |
| **Below Mobile Health Bar Scale** | Slider (1–5) | Escala da barra de vida abaixo do mobile. |
| **Automatically Open Health Bars For Last Attack** | Checkbox | Abre barra de vida ao atacar. |
| **Update One Bar As Last Attack** | Checkbox | Usa uma única barra para o último alvo. |
| **Hidden Player Opacity** | Slider (0–100) | Opacidade de jogadores em stealth. |
| **Hidden Player Hue** | Color picker | Cor dos jogadores hidden. |
| **Regular Player Opacity** | Slider | Opacidade de jogadores normais. |
| **Auto Follow Distance** | Slider (0–10) | Distância de auto-follow. |
| **Disable Auto Follow** | Checkbox | Desactiva auto-follow com Alt (ou tecla configurada). |
| **Disable Mouse Interactions For Overhead Text** | Checkbox | Clicks não interagem com texto overhead. |
| **Override Party Member Hues** | Checkbox | Cor custom para membros do grupo. |
| **Show Target Indicator** | Checkbox | Indicador de alvo. |

## Misc (TazUO)

| Opção | Tipo | Descrição |
|-------|------|-----------|
| **Disable System Chat** | Checkbox | Desactiva o chat de sistema. |

| **Buff Gump Hue** | Color picker | Cor do gump de buffs. |

| **Size** | Slider (1–25) | Tamanho do indicador. |
| **Spell Icon Scale** | Slider (50–300%) | Escala dos ícones de magias. |
| **Display Matching Hotkeys On Spell Icons** | Checkbox | Mostra o atalho em cada ícone. |
| **Hotkey Text Hue** | Color picker | Cor do texto do atalho. |
