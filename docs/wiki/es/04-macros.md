# Macros

Gestión de **macros**: crear, borrar y configurar. Subsecciones: **Nueva macro**, **Borrar macro**, **Macros** (lista).

---

## Nueva macro

Botón que abre el diálogo para **crear una nueva macro**. Se pide un nombre; la macro aparece en la lista y se puede configurar (atajo, acciones).

---

## Borrar macro

Botón para **borrar la macro seleccionada**. Se pide confirmación antes de borrar.

---

## Macros (lista)

Lista de todas las macros del perfil. Al seleccionar una, el panel derecho muestra nombre, atajo y secuencia de acciones. Cada macro se puede editar.
