# Barras de cooldown

Opciones de **barras de cooldown** (TazUO): posición y condiciones de aparición.

| Opción | Tipo | Descripción |
|--------|------|-------------|
| **Barras de cooldown personalizadas** | Área | Sección de barras de cooldown. |
| **Posición X / Y** | Campo numérico | Posición en pantalla (píxeles). |
| **Usar última posición de la barra** | Checkbox | Guardar y usar la última posición al arrastrar. |
| **Condiciones** | Área | Condiciones para cuándo aparece la barra. |
| **Añadir condición** | Botón | Añade una condición; reabre Modern Options en Cooldowns. |
