# General

Opciones generales del cliente, organizadas en subsecciones: **General**, **Móviles**, **Gumps y contexto**, **Misc**, **Terreno y estáticos**.

---

## Subsección: General

| Opción | Tipo | Descripción |
|--------|------|-------------|
| **Resaltar objetos bajo el cursor** | Checkbox | Cuando está activo, los objetos bajo el cursor se resaltan. Ayuda a identificar objetivos e ítems. |
| **Activar pathfinding** | Checkbox | Activa el pathfinding automático: el personaje esquiva obstáculos al clicar para moverse. |
| **Usar Shift para pathfinding** | Checkbox | (Requiere pathfinding.) Solo usa pathfinding al mantener **Shift** al clicar. Sin Shift, movimiento en línea recta. |
| **Un clic para pathfinding** | Checkbox | (Requiere pathfinding.) Un solo clic inicia el movimiento con pathfinding. |
| **Siempre correr** | Checkbox | El personaje corre por defecto. |
| **Excepto si está oculto** | Checkbox | (Requiere Siempre correr.) Corre siempre excepto en stealth/hidden. |
| **Abrir puertas automáticamente** | Checkbox | Abre puertas al pasar. |
| **Abrir puertas durante pathfinding** | Checkbox | (Requiere pathfinding.) Abre puertas en el camino durante el pathfinding. |
| **Abrir cadáveres automáticamente** | Checkbox | Abre cadáveres al acercarse. |
| **Distancia de apertura de cadáver** | Slider (0–5) | Distancia en tiles a la que se abren los cadáveres. |
| **Omitir cadáveres vacíos** | Checkbox | No abre cadáveres ya vacíos. |
| **Opciones de apertura de cadáver** | Combo | Cuándo no abrir: **Ninguno**, **No si hay objetivo**, **No si oculto**, **Ambos**. |
| **Sin color para objetos fuera de alcance** | Checkbox | Objetos fuera de alcance se dibujan en gris. |
| **Activar Sallos easy grab** | Checkbox | Modo de agarrar ítems estilo Sallos. No recomendado con grid containers. |
| **Mostrar contenido de casas** | Checkbox | (Según versión.) Muestra contenido de casas en mapa o contexto. |
| **Movimiento suave de barcos** | Checkbox | (Según versión.) Movimiento de barcos más suave. |

| **Mostrar cuando** | Combo | **Siempre**, **Menos de 100%**, **Smart**. |
| **Resaltar móviles envenenados** | Checkbox | Los móviles envenenados se resaltan con un color. |
| **Color de resalte** (veneno) | Color picker | Color para envenenados. |
| **Resaltar móviles paralizados** | Checkbox | Móviles paralizados resaltados. |
| **Color de resalte** (parálisis) | Color picker | Color para paralizados. |
| **Resaltar móviles invulnerables** | Checkbox | Móviles invulnerables resaltados. |
| **Color de resalte** (invul) | Color picker | Color para invulnerables. |
| **Mostrar nombres de móviles entrantes** | Checkbox | Muestra el nombre de móviles al entrar en pantalla. |
| **Mostrar nombres de cadáveres entrantes** | Checkbox | Muestra el nombre de cadáveres al entrar. |
| **Mostrar aura bajo los pies** | Combo | **Desactivado**, **Modo guerra**, **Ctrl+Shift**, **Siempre**. |
| **Color personalizado para grupo** | Checkbox | Aura del grupo con color personalizado. |
| **Color de aura del grupo** | Color picker | Color. |

## Subsección: Gumps y contexto

|--------|------|-------------|

| **Requerir Alt para mover gumps** | Checkbox | Solo se pueden arrastrar gumps con **Alt**. |
| **Cerrar todo el grupo anclado con clic derecho** | Checkbox | Clic derecho en un gump anclado cierra todo el grupo. |

| **Usar gump de estado antiguo** | Checkbox | Usa el gump de estado antiguo. |

| **Fondo negro** | Checkbox | (Con barras modernas.) Fondo negro en barras de vida. |
| **Guardar barras de vida al salir** | Checkbox | Mantiene posiciones al salir y volver. |
| **Cerrar barras de vida cuando** | Combo | **Desactivado**, **Fuera de alcance**, **Muerto**, **Ambos**. |
| **Grid Loot** | Combo | **Desactivado**, **Solo grid loot**, **Grid loot y contenedor normal**. |
| **Requerir Shift para menús de contexto** | Checkbox | Menús de contexto solo con **Shift**. |
| **Requerir Shift para dividir pilas** | Checkbox | Dividir pilas requiere **Shift**. |
