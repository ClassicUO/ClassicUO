# Barra de acción

Barra de acciones personalizable (botones para hechizos, ítems, macros).

| Qué encuentras | Qué hace |
|----------------|----------|
| **+ Añadir** | Añade un nuevo botón a la barra. |
| **Botones** | Cada botón se puede configurar para una acción (hechizo, ítem, macro, etc.). |

La barra se puede mover y los botones los define el usuario.
