# Contadores

Opciones de **contadores** (maná, vida, stamina mostrados como números o barras).

| Opción | Tipo | Descripción |
|--------|------|-------------|
| **Activación** | Checkbox | Activar o desactivar el sistema de contadores. |
| **Tipo / posición** | Combo/slider | Dónde y cómo se muestran (barra, número, posición). |
| **Valores** | Slider/campo | Umbrales o valores por defecto. |
