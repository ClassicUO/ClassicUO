# Combate y hechizos

| Opción | Qué hace |
|--------|-----------|
| **Mantener Tab para combate** | Tab mantiene el modo combate activo. |
| **Preguntar antes de atacar** | Pregunta antes de atacar (p. ej. objetivos grises). |
| **Preguntar antes de actos benéficos a asesinos/criminales/grises** | Pregunta antes de hechizos benéficos en esos objetivos. |
| **Formato de hechizo en overhead** | Formato del texto de hechizo en overhead. |
| **Hue de hechizo en overhead** | Color del texto de hechizo en overhead. |
| **Un clic en iconos de hechizo** | Un clic en los iconos (en vez de doble). |
| **Duración de buffs en barra antigua** | Duración en la barra de buffs antigua. |
| **Asignación rápida de atajos a hechizos** | Asignar atajos a hechizos rápido. |
| **Ctrl+Alt+clic en icono** | Abre gump para definir atajo en ese icono. |
| **Inocente / Benéfico / Amigo / Dañino / Criminal / Neutral / Atacable / Asesino / Enemigo** | Hue por notoriedad para hechizos y objetivos. |
| **Formato de overhead de hechizo** | Placeholders: p. ej. `{power}`, `{spell}`. |
